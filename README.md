## Language Learning Platform

A modern and interactive Language Learning Platform developed using **HTML5, CSS3, Bootstrap 5, PHP, MySQL, and The platform is designed to help users learn multiple languages through structured lessons, vocabulary and grammar modules, quizzes, assessments, and progress tracking. It provides a user-friendly learning environment where learners can improve their language skills while monitoring their performance and achievements.

The platform includes separate **User** and **Admin** dashboards. Users can explore different languages, access lessons, take quizzes, view their progress, and participate in assessments. Administrators can manage languages, lessons, quiz questions, assessment questions, and user records through a dedicated admin panel.

## How It Works

1. Users register and log in to the platform.
2. Users select a language they want to learn.
3. Lessons are displayed based on the selected language.
4. After completing lessons, users can attempt language-specific quizzes.
5. Quiz scores and attempts are recorded and displayed in the progress section.
6. Users can take a final assessment quiz to evaluate their overall language proficiency.
7. Progress and rankings are reflected in the leaderboard.
8. Administrators manage languages, lessons, quizzes, assessments, and users through the admin dashboard.
